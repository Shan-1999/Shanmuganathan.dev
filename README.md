# 🌌 shanmuganathan.dev — Portfolio Website

Personal portfolio of **Shanmuganathan S**, Software Engineer based in Chennai, India.

> Built with pure HTML, CSS & JavaScript — no frameworks, no build tools. Just a single file and the cosmos.

---

## ✨ Features

- **Cosmic space background** — procedural star field (500+ stars), animated nebula clouds, shooting stars, and a Milky Way streak rendered on canvas
- **Glassmorphism UI** — frosted glass cards with `backdrop-filter` blur, inset shine borders, and smooth hover transitions
- **Fully responsive** — mobile-first layout, floating pill navbar
- **Zero dependencies** — no npm, no bundler, no frameworks. One `index.html` file
- **Smooth scroll navigation** — anchor-linked sections
- **Download Resume** — direct PDF download button

## 🚀 Running Locally

No setup required. Just open the file:

```bash
git clone https://github.com/Shan-1999/shan-1999.github.io.git
cd shan-1999.github.io
open index.html   # macOS
# or just double-click index.html on Windows
```

Or serve it locally for proper font loading:

```bash
npx serve .
# then open http://localhost:3000
```
